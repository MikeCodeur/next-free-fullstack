## Comprendre

Nous allons utiliser shadcn pour créer un formulaire de login.

En plus de gérer les composants, shadcn permet de gerer des block plus complexes.

Par exemple

- [https://ui.shadcn.com/blocks](https://ui.shadcn.com/blocks)

Pour récupérer un block

```bash
pnpm dlx shadcn@latest add login-03
```

📑 Le liens vers la doc [https://fr.reactjs.org/docs/getting-started.html](https://fr.reactjs.org/docs/getting-started.html)
